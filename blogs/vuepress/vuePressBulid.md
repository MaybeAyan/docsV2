## 前期准备

    框架使用 VuePress，主题使用 reco 1.x 版本 

## 新建项目

    一、安装

    npm install @vuepress-reco/theme-cli -g

    或者 yarn global add @vuepress-reco/theme-cli


    二、创建项目

    theme-cli init


    三、安装依赖

    yarn 或者 npm install


    四、运行项目

    yarn dev 或者 npm run serve

## 修改一些配置信息

    主要是在 .vuepress/config.js 下进行配置相关信息
